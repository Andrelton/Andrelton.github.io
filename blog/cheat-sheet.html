Eric's & Ellis's Fun Ruby Enumerable Cheat Sheet

Name Description Syntax Result
#find Returns the first item for which the block is not false. [1, 2, 3, 4, 5].find { |item| item.even? } #=> 2
#find_all Returns an array of items for which the block is not false. [1, 2, 3, 4, 5].find_all { |item| item.even? } #=> [2, 4]
#reject Return an array of all items for which the block return false. [1, 2, 3, 4, 5].reject { |item| item.even? } #=> [1, 3, 5]
#sort_by Returns an array based on sorting by a given block. [1, 2, 3, 4, 5].sort_by { |item| 20 - item } #=> [5, 4, 3, 2, 1]
#min_by Returns item that is the minimum value from a given block. [1, 2, 3, 4, 5].sort_by { |item| 20 - item } #=> 5
#cycle Iterates through enumerable a given number of times. [1, 2, 3, 4, 5].cycle { |item| print item } #=> 1234512345
#count Returns the number of times an item occurs in a given array. [1, 2, 3, 4, 5].count(3) #=> 1
#each_with_object Accepts an object that can be used with a block. [1, 2, 3, 4, 5].each_with_object([]) { |i, a| a << [i] } #=> [[1], [2], [3], [4], [5]]
#grep Returns an array of all items that match the pattern.
If optional block is supplied, items are passed to block before returning.		 [1, 2, 3, 4, 5].grep(2..4) { |item| item + 1 } #=> [3, 4, 5]
#partition Returns two arrays, the first in which items evaluate to true,
the second in which items evaluate to false.		 [1, 2, 3, 4, 5].partition { |item| item.even? } #=> [[2, 4], [1, 3, 5]]
